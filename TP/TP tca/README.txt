simulation.py: Script Python simulant une enchère multi-agent.
- 2 fournisseurs, 2 billets chacun (avec réserve).
- 10 acheteurs, budgets, offres initiales et incréments random.
- 3 tours d'offres, retrait si budget dépassé.
- Clôture avec annonce des ventes.

Exécuter:
  python part1.py -----> sans coalition.
  puthon part2.py -----> avec coalition.
